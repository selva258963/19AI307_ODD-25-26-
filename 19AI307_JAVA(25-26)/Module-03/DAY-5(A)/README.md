## AIM:
To write a Java program that demonstrates the use of non-static inner classes, where the inner class accesses methods to display a message based on user input.

## ALGORITHM :

6. Call the showMessage() method of the inner class with the name as argument.
7. Display the greeting message generated by the inner class.

## SOURCE CODE:

```
import java.util.*;

public class OuterClass {

    class InnerClass {
        void showMessage(String name) {
            System.out.println("Hello, " + name + "! This message is from the Inner Class.");
        }
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String name = sc.nextLine();

        OuterClass outer = new OuterClass();
        OuterClass.InnerClass inner = outer.new InnerClass();
        inner.showMessage(name);
    }
}
```
